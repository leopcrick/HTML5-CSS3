Trabalhando com Listas

Listas Ordenadas

Veja a lista numerada

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Podemos também numerar a lista alfabeticamente

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Fazemos essas alterações com o comando type="a", para organizar em ordem alfabética com letras minúsculas, type="A" para maiúsculas, type="i" para algarismos romanos minúsculos e type="I" para romanos maiúsculos. O padrão é type="1" em que é aplicada a ordem numérica normal.

Também é possível fazer a lista começando de outro ponto com o comando start="4", veja:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

Listas Não Ordenadas

Agora veja a lista sem numeração

  • Acordar
  • Ligar para o João
  • Tomar café
  • Escovar os dentes

Podemos também selecionar o ícone da listagem adicionando o comando type=""

  • type="circle"
  • type="square"
  • type="disc"

Comandos legais

Escrevendo li*7, você cria 7 linhas com o comando li. Também é possível digitando ol>li*4, onde será criado o comando ol com as 4 li dentro. O mesmo vale para qualquer comando aparentemente.

Apertando Ctrl+Enter você começa um nova linha, independente de onde o cursos esteja.
